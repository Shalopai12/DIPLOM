О проекте
Платформа решает проблему разрозненности информации о кофейнях (меню, акции, расписание, контакты) и предоставляет единое пространство для взаимодействия гостей, сотрудников и управляющей компании.

Ключевые возможности:

Онлайн‑меню с актуальными ценами, составом и фото напитков и блюд.

Фильтры по типам напитков (эспрессо, альтернативные методы, авторские, без кофеина и т.п.).
​

Онлайн‑заказ с собой (pick‑up) и, при необходимости, доставка.

Программа лояльности: личный кабинет с накопительными бонусами, историей заказов и персональными предложениями.

Отображение ближайших кофеен на карте, режим работы и загруженность.
​

Управление акциями, спецпредложениями и новостями сети.

Система отзывов и рейтингов для кофеен и отдельных позиций меню.

Разграничение прав доступа: гость / зарегистрированный пользователь / бариста / управляющий / администратор.

Административная панель для управления меню, акциями, заказами и пользователями.
​
​

Технологический стек
Бэкенд
Python 3.13+ — язык программирования для реализации серверной логики.
​
​

Django 5.2 (включая ORM) — веб‑фреймворк для реализации бизнес‑логики, работы с моделями (кофейни, товары, заказы, пользователи) и маршрутизации.
​
​

PostgreSQL 17 — реляционная база данных для надежного хранения данных о меню, заказах, пользователях и транзакциях.
​
​

Django Admin — встроенная панель администратора для управления меню, акциями, кофеенами, заказами и персоналом.

Фронтенд
HTML5 — для построения структуры страниц (главная, меню, кофеени на карте, личный кабинет, корзина, акции).
​
​

CSS3 — для стилизации, адаптивной верстки под мобильные устройства и фирменного стиля сети кофеен.
​
​

JavaScript — для интерактивных элементов (динамическое обновление корзины, выбор кофеен, фильтрация меню).
​
​

AJAX — для асинхронного обновления контента без перезагрузки страницы (добавление в корзину, применение промокода, обновление статуса заказа).
​
​

Bootstrap — для адаптивной сетки, готовых UI‑компонентов и ускорения верстки.
​
​

Безопасность
CSRF‑токены — защита форм заказа, авторизации и управления аккаунтом от межсайтовых атак.
​
​

Валидация форм на стороне клиента и сервера (регистрация, вход, платежные данные, отзывы, создание акций).
​
​

Защита от SQL‑инъекций за счет использования Django ORM при работе с базой данных.
​
​

Безопасное хранение паролей и конфиденциальной информации клиентов в соответствии с рекомендациями Django.
